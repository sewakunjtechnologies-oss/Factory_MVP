# Mobile Factory Owner QA

Use this checklist on Android after installing or opening the Capacitor build.

1. Install/open Android app.
2. Confirm mobile home opens instead of the desktop sidebar dashboard.
3. Tap Create PO.
4. Search for a product/fabric category.
5. Enter quantity and delivery month.
6. Create PO.
7. Confirm fabric check result appears.
8. Test one sufficient-fabric category.
9. Test one shortage category.
10. Confirm shortage popup/card shows required, available, and shortage meters.
11. Snooze shortage reminder for 4 hours.
12. Reopen app and confirm reminder summary.
13. Open a fabric-ready PO.
14. Tap Move to Cutting.
15. Confirm the transition.
16. Move through cutting completion.
17. Move through stitching completion.
18. Record repair and alteration quantities.
19. Move through packing completion.
20. Dispatch a partial quantity.
21. Dispatch remaining quantity.
22. Confirm completion blocks if unresolved quantity remains.
23. Tap Ask AI.
24. Ask AI to update a stage.
25. Confirm AI asks missing information.
26. Confirm AI speaks or displays the result.
27. Ask another question immediately.
28. Generate PDF through AI.
29. View May historical POs after import is applied.
30. Edit one May PO from the existing web edit path if needed.
31. View June historical POs after import is applied.
32. Edit one June PO from the existing web edit path if needed.
33. Open desktop browser and confirm the existing web app remains unchanged.
34. Open desktop Create PO and confirm it still has separate Category and Fabric fields.
35. Confirm mobile Create PO has only one Product / Fabric Category field.
