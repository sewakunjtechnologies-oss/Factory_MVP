# Factory Control Frontend

React + TypeScript control panel for the factory execution backend.

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- TanStack Query
- Axios
- React Router
- Zustand

## Run

```bash
npm install
npm run dev
```

The app uses `VITE_API_BASE_URL` when present, otherwise it connects to:

```text
http://127.0.0.1:8000/api/v1
```
